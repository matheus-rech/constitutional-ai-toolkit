## üü° Custom Domain (Optional)

If you want to use your own domain (e.g., `constitutional-ai.yourdomain.com`):

### GitHub Pages:
1. Add `CNAME` file with your domain
2. Configure DNS: `CNAME` record pointing to `YOUR_USERNAME.github.io`
3. Enable in Settings ‚Üí Pages ‚Üí Custom domain

### Vercel:
```bash
vercel domains add constitutional-ai.yourdomain.com
```
Then follow DNS instructions

### Netlify:
1. Domain settings ‚Üí Add custom domain
2. Follow DNS configuration instructions
